# ☁️ Weather Vibe Check

A sassy, unserious weather application built with HTML, CSS, and Vanilla JavaScript. This project connects to a real-world API to give you the current weather along with a "vibe check."

## 🚀 Live Demo
Check out the app here: [https://arushi-1706.github.io/weather-vibe-check/](https://arushi-1706.github.io/weather-vibe-check/)

## ✨ Features
- **Real-time Data:** Fetches live weather from the WeatherAPI.
- **Dynamic Styling:** The background gradient changes based on the temperature (from "Deadly Laser" heat to "Ice Age" cold).
- **Sassy Logic:** Provides an "unserious" commentary on the current conditions.
- **Visual Assets:** Automatically displays weather icons (sun, moon, rain) provided by the API.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3
- **Scripting:** JavaScript (ES6+)
- **API:** [WeatherAPI](https://www.weatherapi.com/)
